Synopsis

xsngine is a flexible open-source game engine targeting ridiculously high framerates with loads of pretty things.
Runs natively on 32/64 bit Windows, Mac and Linux using SDL2 and OpenGL for maximum portability.

The primary goal of the project is to build a powerful engine that can create games which feel great, rather than creating a one size fits all "artist's toolkit"
Drawing inspiration from iD Tech and competitive shooters, performance is a #1 priority whilst bringing modern rendering techniques to the table.

Features

Below is a list of planned, incomplete and already implemented features.

  • Deferred rendering (diffuse, specular, normal) with anti-aliasing
  • Physically Based Rendering (PBR)
  • Shadow mapping
  • Particle system
  • Real world ighting phenomena (fog, high dynamic range rendering, bloom, scotopic luminance, depth of field + bokeh)
  • Import/export tools for modelling software -> XMF (xsngine model format)
  • Client/server networking model ala iD Tech 3
  • High precision, low latency input controls with multiple input methods
